# znajdzkorty-screenshots

Public screenshots host for [dev-oktal/znajd-korty-app](https://github.com/dev-oktal/znajd-korty-app) PR descriptions.

The main repo is private, so its `raw.githubusercontent.com` URLs return 401 to GitHub's image proxy when rendering PR descriptions. This repo provides public mirrors for inline images.

Authoritative versions of these screenshots also live at `docs/superpowers/screenshots/` inside the main repo (referenced via clickable links as fallback).
